# QQ聊天 (aurora-app-qq)

**包名**: `im.polaris.qq`

**版本**: 1.2.0

**最低 Brain 版本**: >=4.1.0

负责 QQ 消息的接收、发送和会话目标管理。消息会被真实的人看到, 所以后续内核通常需要结合上下文判断节奏、语气与是否回复; 短时间内同一会话的多条消息往往应综合理解后再处理。如果需要分多条发送, 应调用多次发送命令, 而不是把多条内容拼进同一个 text 字段。

---

## 指令

### `send_qq_message` -- 发送群/会话消息

向一个 QQ 群或会话发送文本消息; 消息发出后会被对方或群成员看到。

| 参数         | 类型   | 必填 | 说明                                                     |
| ------------ | ------ | ---- | -------------------------------------------------------- |
| `session_id` | string | 是   | 群号或 QQ 号                                             |
| `text`       | string | 是   | 一条实际发出的消息内容; 若要分多条发送, 应多次调用该工具 |

| 返回字段       | 类型    | 说明           |
| -------------- | ------- | -------------- |
| `success`      | boolean | 是否发送成功   |
| `delivered_at` | number  | 实际发送时间戳 |

### `send_qq_private_message` -- 发送私聊消息

向指定 QQ 用户发送私聊消息; 私聊内容只有双方可见。

| 参数      | 类型   | 必填 | 说明                                                     |
| --------- | ------ | ---- | -------------------------------------------------------- |
| `user_id` | string | 是   | 目标 QQ 号                                               |
| `text`    | string | 是   | 一条实际发出的私聊消息; 若要分多条发送, 应多次调用该工具 |

| 返回字段  | 类型    | 说明         |
| --------- | ------- | ------------ |
| `success` | boolean | 是否发送成功 |

### `at_user_in_group` -- @群成员并发送消息

在群里 @ 某个用户并发送消息; 被 @ 的用户会收到更强提醒。

| 参数       | 类型   | 必填 | 说明                                                   |
| ---------- | ------ | ---- | ------------------------------------------------------ |
| `group_id` | string | 是   | 群号                                                   |
| `user_id`  | string | 是   | 被 @ 的 QQ 号                                          |
| `text`     | string | 是   | 一条实际发出的群消息; 若要分多条发送, 应多次调用该工具 |

| 返回字段  | 类型    | 说明         |
| --------- | ------- | ------------ |
| `success` | boolean | 是否发送成功 |

---

## 使用

在 `legacy/config.yml` 的 `apps` 节点下添加或启用 `qq`:

```yaml
apps:
  qq:
    enabled: true
    startup:
      enable_listener: true
    # //其他应用//
```

- `enabled: true` -- 启用该应用
- `startup.enable_listener: true` -- 启动 QQ 消息监听器, 接收消息
